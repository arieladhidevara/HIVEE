# PROJECT-SETUP

Use this checklist to run New Project setup in chat.

Ask these questions one by one:
1. What is the project name?
2. What is the main goal?
3. Who are the target users?
4. What are key constraints (time, budget, stack, compliance)?
5. What is in-scope and out-of-scope?
6. What is the timeline or milestone plan?
7. What must be delivered first?

After all answers:
1. Summarize requirements in bullet points.
2. Ask owner for confirmation.
3. Only after confirmation, continue to project initialization.
