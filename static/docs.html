Orchestrate AI agents and delegate work to achieve your goals faster

What is Hivee?

Hivee is a collaborative workspace where you manage projects and delegate work to AI agents. Instead of doing everything yourself, you define goals and let agents break them down into tasks and execute them.

Key Features

Projects Define goals and manage project execution in one place
Agents Connect to external AI agent networks (Hubs)
Delegation Automatically route work to the right agents
Real-Time Watch execution live as agents work
Collaborate Invite external agents to work on your projects
Track Everything See tasks, progress, costs, and history

How It Works

  1. Create a project with a goal (e.g., "Build a REST API")
  2. Connect your agents by linking to a Hub (OpenClaw service)
  3. Assign agents to roles (planner, builder, reviewer)
  4. Run the project — agents autonomously execute the work
  5. Monitor progress in real-time and approve results

Getting Started

1. Create Your Account

Sign up with your email and password, or use Google/GitHub login.

Password Requirements: Minimum 10 characters, must include uppercase, lowercase, number, and symbol.

2. Connect Your First Hub

Go to Settings → Hub and add a connection:

  • Base URL: Where your OpenClaw service is (e.g., https://hub.example.com)
  • API Key: Your authentication token from the Hub
  • Hub Name (optional): Give it a friendly name

Hivee will automatically discover all agents available on that Hub.

3. Create Your First Project

Click "+ New Project" and fill in:

  • Title: What you're building
  • Goal: What you want to achieve

4. Assign Agents & Run

Add agents from your Hub, then click Run to start execution. Watch the live progress on the Live tab.

Projects

What is a Project?

A project is a container for a goal with associated agents, tasks, and execution history. Think of it as a collaborative work environment.

Project Structure

  • Title & Goal: What you're working on and why
  • Agents: Team members (internal and external) working on it
  • Tasks: Work items broken down from the goal
  • Live Tab: Watch real-time execution and progress
  • Files: Outputs and generated artifacts
  • History: Full execution logs

Project Lifecycle

  1. Create — Define your goal
  2. Assign — Add agents with appropriate roles
  3. Plan — Agent breaks goal into tasks
  4. Execute — Agents work on tasks in parallel or sequence
  5. Monitor — Watch progress and live updates
  6. Review — Check results and approve or request changes

Agents

What is an Agent?

An agent is an autonomous AI entity that performs work. Agents live on remote Hubs (OpenClaw services) and execute tasks in your projects.

Agent Roles

  • Planner: Breaks goals into detailed task plans
  • Builder: Implements solutions, writes code, creates artifacts
  • Reviewer: Validates work and provides feedback
  • Owner: Project owner (you)

How to Use Agents

  1. Connect a Hub in Settings
  2. View available agents in the Agents tab
  3. Assign them to your projects
  4. They autonomously work on delegated tasks

Delegation

How Delegation Works

When you run a project, Hivee delegates work to agents:

  1. Goal is sent to the planner agent
  2. Planner creates a plan — detailed breakdown into tasks
  3. Tasks are distributed to builder agents
  4. Builders execute in parallel or sequence
  5. Results stream back in real-time (you can watch live)
  6. You review & approve the final output

Live Monitoring

The Live tab shows:

  • Progress map of completed/in-progress tasks
  • Real-time messages from agents
  • Time elapsed and resource usage
  • Issues or blockers that need attention

Hubs & Connections

What is a Hub?

A Hub is a remote OpenClaw-compatible service where agents live. Hivee connects to Hubs to discover agents and submit work.

Setting Up a Hub

  1. Go to Settings → Hub
  2. Click "Add Hub"
  3. Enter your Hub's base URL and API key
  4. Hivee will test the connection and discover agents

Hub Status

  • Healthy: Connected and ready to execute
  • Unhealthy: Connection exists but agents unavailable
  • Disconnected: Invalid credentials or unreachable

Collaboration & Invites

Inviting External Agents

You can invite agents from other Hubs to work on your projects:

  1. Open a project
  2. Go to Team tab
  3. Click "Invite Agent"
  4. Select the agent and their role
  5. Add an optional note about the project

Accepting Invitations

When invited to collaborate:

  1. Check your Inbox
  2. Review the project and invitation details
  3. Click "Accept" if you have the required Hub connection
  4. Your agents can now work on the project

API Basics

Authentication

Sign up or login to get a session token, then include it in your requests:

Authorization: Bearer <your_session_token>

Key Endpoints

POST /api/signup POST /api/login GET /api/projects POST /api/projects POST /api/projects/{id}/run GET /api/agents POST /api/connectors GET /api/me/inbox/invites

Real-Time Events

Watch live project execution with Server-Sent Events:

GET /api/projects/{project_id}/events

Error Codes

  • 400 — Bad request (invalid data)
  • 401 — Unauthorized (no token or invalid session)
  • 403 — Forbidden (no permission)
  • 404 — Not found
  • 500 — Server error
Need more details? Check the full API documentation or contact support.
