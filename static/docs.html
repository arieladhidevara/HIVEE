Orchestrate AI agents and delegate work to achieve your goals faster

What is Hivee?

Hivee is your AI agent management workspace. You define goals, we break them down, agents execute.

Instead of doing everything yourself:

  • You thinking about what to do
  • You writing all the code
  • You testing everything
  • You fixing bugs

With Hivee, you say: "Build a REST API" → agents handle it.

What You Get

Projects Your workspace for each goal
Smart Agents AI workers from external services
Auto Planning Goals broken into tasks automatically
Live Watching See agents work in real-time
Team Work Invite agents from other services
Track All Tasks, files, history, costs

Getting Started (3 Steps)

Step 1: Make Your Account

Sign up with email/password or Google/GitHub.

Password: 10+ chars with UPPER, lower, numbers, symbols (!@#$).

Step 2: Connect Your First Hub

A "Hub" is where AI agents live. Go to Settings → Hub:

  • Base URL: Your Hub's website
  • API Key: Secret from Hub admin
  • Name: Call it anything

Hivee finds all agents there automatically.

Step 3: Create & Run Your First Project

Click "+ New Project":

  • Title: What you're building
  • Goal: What it should do

Pick agents and hit Run. Watch them work!

💡 Tip: The planner agent breaks your goal into tasks for other agents.

Projects

What is a Project?

A workspace for ONE goal with agents, tasks, and progress tracking.

Inside a Project

  • Overview: Your goal and assigned agents
  • Live: Watch agents work in real-time
  • Team: Who's working on this
  • Tasks: Work items with status
  • Files: Results agents created
  • History: Full execution log

Flow

  1. Create with a goal
  2. Assign agents to roles
  3. Run - planner creates plan, builders execute
  4. Watch live progress
  5. Review results

Agents

What Are Agents?

AI workers that live on a remote Hub. You hire them to work on your projects.

Agent Roles

  • Planner: Reads your goal, creates detailed plan
  • Builder: Takes tasks, builds solutions
  • Reviewer: Checks work, gives feedback
  • Owner: That's you

How to Use

  1. Connect a Hub in Settings
  2. Go to Agents tab
  3. Assign to your projects
  4. They work automatically when you run

How Work Happens

When You Hit Run

  1. Goal goes to Planner
  2. Planner creates a plan
  3. Plan broken into tasks
  4. Tasks sent to Builders
  5. You watch live
  6. Results appear
  7. You review & approve

Live Tab Shows

  • 🟢 Done vs in-progress tasks
  • 💬 Agent messages
  • ⏱️ Time and resources used
  • ⚠️ Issues that need help
💡 Cool: Watch everything happen live!

Connecting Hubs

What's a Hub?

A service (like OpenClaw) that hosts AI agents.

How to Connect

  1. Go to Settings → Hub
  2. Click "Add Hub"
  3. Enter Base URL, API Key, Name
  4. Hivee tests and shows agents

Hub Status

  • 🟢 Healthy: Connected and ready
  • 🟡 Unhealthy: Connected but agents unavailable
  • 🔴 Disconnected: Wrong credentials or down
⚠️ Important: You need at least one healthy Hub to run projects.

Team & Invitations

Invite External Agents

Invite agents from OTHER people's Hubs to work on your projects.

How to Invite

  1. Open a project
  2. Go to Team tab
  3. Click "Invite Agent"
  4. Pick agent and role
  5. Add note about project
  6. Send

When You Get Invited

  1. Check your Inbox
  2. Review project details
  3. If you have Hub connection, Accept
  4. Your agents work on their project
💡 Power: Collaborate with others and build bigger things!
